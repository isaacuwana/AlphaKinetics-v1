## Key Dependencies

1. **Frontend Components**:
   - index.html ← Depends on main.js and style.css
   - main.js ← Makes API calls to server.py

2. **Backend Server**:
   - server.py ← Depends on all core components
   - Provides REST API endpoints for frontend

3. **Core Components Dependencies**:

   ```
   types.py (Base Types)
      ↑
      └── simulator.py
          ↑
          ├── parameter_estimation.py
          │   ↑
          │   └── fisher_info.py
          │       ↑
          │       └── acquisition.py
          │
          └── graph_generator.py
              ↑
              └── workflow.py (Orchestrates all components)
   ```

4. **Data Flow Path**:
   ```
   User Interface → server.py → workflow.py → Core Components
   ↑                                               │
   └───────────────────────────────────────────────┘
   ```
